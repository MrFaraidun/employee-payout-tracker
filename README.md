# Employee Payout Tracker

A minimalist, dark-first fintech dashboard for logging and managing employee payouts.

## Tech Stack & Highlights
- **Backend:** Laravel (Spatie Laravel Permission, SQLite/PostgreSQL)
- **Frontend:** Vue 3 (Composition API, Inertia.js)
- **Styling:** Tailwind CSS, Glassmorphism, Double-Bezel Bento grids
- **Navigation:** Responsive Left Slide-Out Drawer (Mobile & Tablet)
- **UI Details:** Custom select dropdown components, custom date picker filters
- **Status:** Production Ready (Fly.io)
